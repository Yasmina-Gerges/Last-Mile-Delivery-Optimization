# Last-Mile Delivery Optimization Using Simulation and MILP

## Project Overview

This project combines agent-based simulation and mathematical optimization to improve last-mile delivery operations.

The system models:
- Drivers
- Restaurants
- Customers
- Dynamic traffic conditions
- Stochastic customer demand

Optimization decisions are generated using Mixed Integer Linear Programming (MILP) through OR-Tools.

## Technologies

- AnyLogic
- Python
- OR-Tools
- Java

## Key Results

- 9% reduction in delivery costs
- 1.9-minute reduction in average delivery time
- Outperformed nearest-rider dispatching under varying traffic and demand conditions
